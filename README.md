# -Multi-user-MIMO-OFDMA-Channel-Capacity-Enhancement-with-constant-bandwidth
## Overview
This project investigates the enhancement of wireless channel capacity using Multi-User MIMO and OFDMA techniques while maintaining constant bandwidth and transmit power.

## Objectives
- Improve channel capacity
- Increase throughput
- Analyze BER performance
- Evaluate beamforming techniques

## Tools
- MATLAB (Simulation and Performance Analysis)
## Key Results 
- MIMO-OFDMA significantly improves channel capacity over SISO, SIMO, MISO, and SU-MIMO systems 
- Channel capacity increases as SNR increases (strong positive correlation)
- Higher antenna configurations (2×2 → 5×5) increase capacity from ~6.25 to ~14.25 bits/s/Hz at 20 dB SNR  
- MIMO-OFDMA outperforms standalone MIMO and OFDMA in all tested scenarios  
- Beamforming reduces interference and improves signal quality and directionality  
- Beamforming achieves lower Bit Error Rate (BER) compared to non-beamforming systems  
- BER decreases with increasing SNR, improving overall link reliability  
- Overall gains: higher spectral efficiency, better reliability, and improved capacity without extra bandwidth
